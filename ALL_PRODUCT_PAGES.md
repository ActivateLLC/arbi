## 🔑 Google Ads API Setup

Ready to configure Google Ads! Add these to `apps/api/.env`:

```bash
# Google Ads API Credentials
GOOGLE_ADS_DEVELOPER_TOKEN=your_developer_token
GOOGLE_ADS_CLIENT_ID=your_oauth_client_id
GOOGLE_ADS_CLIENT_SECRET=your_oauth_client_secret
GOOGLE_ADS_REFRESH_TOKEN=your_refresh_token
GOOGLE_ADS_CUSTOMER_ID=123-456-7890  # Your Google Ads account ID
```

### How to Get These Credentials

1. **Developer Token**: https://developers.google.com/google-ads/api/docs/get-started/dev-token
2. **OAuth Credentials**: https://console.cloud.google.com/apis/credentials
3. **Refresh Token**: Run OAuth flow with the API client
4. **Customer ID**: Found in Google Ads top right corner
