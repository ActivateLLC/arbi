# Google Ads API Setup Guide

To enable the "Real Money" mode with Google Ads, you need to configure the following environment variables in Railway (and locally in `apps/api/.env`).

## 1. Get Credentials

You already have `client_secret.json` in the root folder. This contains your `CLIENT_ID` and `CLIENT_SECRET`.

### Required Variables:

1.  **GOOGLE_ADS_CLIENT_ID**: Found in `client_secret.json` (under `web.client_id` or `installed.client_id`).
2.  **GOOGLE_ADS_CLIENT_SECRET**: Found in `client_secret.json` (under `web.client_secret` or `installed.client_secret`).
3.  **GOOGLE_ADS_DEVELOPER_TOKEN**: You must get this from your [Google Ads Manager Account](https://ads.google.com/home/tools/manager-accounts/).
    *   Go to **Tools & Settings** > **Setup** > **API Center**.
4.  **GOOGLE_ADS_CUSTOMER_ID**: Your Google Ads Customer ID (xxx-xxx-xxxx) without dashes.
5.  **GOOGLE_ADS_REFRESH_TOKEN**: Generated using the script below.

## 2. Generate Refresh Token

We have updated the `generate-refresh-token.js` script to automatically use your `client_secret.json`.

1.  Open a terminal in the root directory.
2.  Run the script:
    ```bash
    node generate-refresh-token.js
    ```
3.  Follow the instructions:
    *   Copy the URL provided.
    *   Paste it into your browser.
    *   Authorize the app.
    *   Copy the code provided by Google.
    *   Paste the code back into the terminal.
4.  The script will output your `REFRESH_TOKEN`.

## 3. Configure Railway

1.  Go to your [Railway Dashboard](https://railway.app/).
2.  Select the `arbi` project.
3.  Select the `api` service (or the main service if monorepo).
4.  Go to **Variables**.
5.  Add the following variables:

| Variable Name | Value |
| :--- | :--- |
| `GOOGLE_ADS_CLIENT_ID` | (From client_secret.json) |
| `GOOGLE_ADS_CLIENT_SECRET` | (From client_secret.json) |
| `GOOGLE_ADS_DEVELOPER_TOKEN` | (From Google Ads Interface) |
| `GOOGLE_ADS_CUSTOMER_ID` | (Your Account ID) |
| `GOOGLE_ADS_REFRESH_TOKEN` | (Generated by script) |

## 4. Verify

Once these are set, the Autonomous Engine will automatically start creating real ad campaigns for profitable products it finds.
